Реализация протокола раздедение секрета

Протокол позволяет создать секрет для группы участников.
Секрет можно раcшифровать при наличии БОЛЬШИНСТВА участников.
К сожалению данная схема не предусматривает выявление провокатора.
А также требуется честность дилера.

Схема Шамира (https://ru.wikipedia.org/wiki/Схема_разделения_секрета_Шамира):
1) Секрет - свободный секрет многочлена
2) Генерируется многочлен степень N // 2
3) Вычисляется N значений многочлена, что и является ключом
4) Вычисленные значения разлаются участникам
5) N // 2 + 1 участник могут восстановить секрет

Ключи:
    -n - количество участников
