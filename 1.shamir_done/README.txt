Реализация протокола Шамира

В качестве коммутативного шифра используется шифр предложенный самим Шамиром (https://studfiles.net/preview/4421392)
Протокол позволяет передавать секрет по открытому каналу за 3 передачи данных по сети

1) Алиса генерирует простое число p, а так же числа da и ea такие, что da*ea = 1 (mod p-1)
2) Алиса шифрует секрет (x1 = secret^ea mod p), отправляет x1 и p Бобу
3) Боб генерирует свои db и eb так что db*eb = 1 (mod p-1).
4) Боб шифрует полученное сообщение (x2 = x1^eb mod p), отправляет x2 Алисе
5) Алиса расшифровывает полученное сообщение (x3 = x2^da mod p), отправляет x3 Бобу
6) Боб расшифровывает полученное сообщение (x4 = x3^db modp), x4 = secret

Ключи:
    -s - секретная строка
